### Simple Discovery Job
A simple job with no external file reference
```json
{
  "id": "simple_1234",
  "action": "ADD",
  "entry": {
    "meta": {"body": "<xml>raw XML metadata<\/xml>"},
    "fields": {
      "name": "Name (1381454934615)",
      "abstract": "some longer text about what we have"
    }
  }
}
```

        "relation": "data",
